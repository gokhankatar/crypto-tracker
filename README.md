## Installation

To clone the repository, run the following command:

```sh
git clone https://github.com/gokhankatar/crypto-tracker

``` 
go to directory

```sh
cd .\crypto-tracker\

``` 
and then install npm:

```sh
npm install -f

``` 
open your browser:

```sh
npm run dev
